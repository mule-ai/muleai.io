## Overview

Repositories are the foundation of Mule's autonomous development capabilities. By adding a repository, you enable Mule to:

- Monitor issues labeled with `mule`
- Create branches and generate code changes
- Submit pull requests with AI-generated solutions
- Respond to code review feedback and comments
- Process repositories on a configurable schedule

## Supported Providers

| Provider | Description |
|----------|-------------|
| **GitHub** | Full integration with GitHub repositories. Requires GitHub token or SSH access. |
| **Local** | Work offline with repositories on your local filesystem. Ideal for testing. |
